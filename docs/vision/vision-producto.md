# Visión del Producto – App Bienestar Universitario UIDE

La aplicación de Bienestar Universitario UIDE tiene como objetivo centralizar y digitalizar los procesos de atención estudiantil, permitiendo a los estudiantes gestionar solicitudes, recibir información institucional y acceder a servicios de apoyo de forma rápida, segura y transparente.

El sistema busca reducir tiempos de atención, mejorar la comunicación entre estudiantes y el personal de Bienestar, y proporcionar información confiable para la toma de decisiones institucionales.

Está dirigida principalmente a estudiantes UIDE, personal de Bienestar Universitario y áreas administrativas relacionadas como Becas, Finanzas y TI.
