# HU-04 – Visualización de solicitudes pendientes

**Como** personal de Bienestar Universitario  
**Quiero** visualizar las solicitudes pendientes  
**Para** priorizarlas y gestionarlas eficientemente  

## Criterios de Aceptación (Gherkin)

Escenario: Visualización de solicitudes pendientes  
Dado que ingreso al panel administrativo  
Cuando accedo al módulo de solicitudes  
Entonces el sistema muestra las solicitudes pendientes de revisión  

Escenario: Visualización del detalle de la solicitud  
Dado que selecciono una solicitud  
Cuando accedo a su detalle  
Entonces el sistema muestra los datos del estudiante y los documentos adjuntos  

Escenario: Descarga de documentos  
Dado que la solicitud contiene documentos adjuntos  
Cuando selecciono un documento  
Entonces el sistema permite su descarga  

Escenario: Filtrado de solicitudes  
Dado que existen múltiples solicitudes  
Cuando aplico filtros  
Entonces el sistema muestra únicamente las solicitudes solicitadas
