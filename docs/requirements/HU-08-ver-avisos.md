# HU-08 – Visualización de avisos institucionales

**Como** estudiante UIDE  
**Quiero** visualizar avisos institucionales  
**Para** mantenerme informado sobre eventos y comunicados

Escenario: Visualización de avisos  
Dado que ingreso a la aplicación

Entonces el sistema muestra los avisos publicados
