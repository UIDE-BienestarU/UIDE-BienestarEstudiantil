# HU-06 – Recepción de notificaciones

**Como** estudiante UIDE  
**Quiero** recibir notificaciones cuando cambie el estado de mi solicitud  
**Para** mantenerme informado oportunamente  

## Criterios de Aceptación (Gherkin)

Escenario: Notificación por cambio de estado  
Dado que tengo una solicitud enviada  
Cuando el estado de la solicitud cambia  
Entonces el sistema envía una notificación al estudiante  

Escenario: Visualización de la notificación  
Dado que recibo una notificación  
Cuando la abro  
Entonces el sistema muestra información resumida del cambio de estado
